## REMOVED Requirements

### Requirement: Branded single-page structure

**Reason:** The standalone Next.js marketing frontend is retired. The repository now maintains only the Vue analysis platform.

**Migration:** Product-marketing requirements may be reintroduced through a future independent capability targeting Vue or a separately deployed website.

### Requirement: Sports-tech visual style

**Reason:** The requirement was scoped to the retired standalone landing page.

**Migration:** The maintained platform visual style remains covered by `swim-analysis-platform-navigation`.

### Requirement: Fixed responsive navigation

**Reason:** The navigation belonged to the retired marketing page.

**Migration:** Business-platform navigation remains covered by `swim-analysis-platform-navigation`.

### Requirement: Hero explains the product in the first viewport

**Reason:** The standalone public marketing hero is no longer part of the supported Web application.

**Migration:** None in this Change.

### Requirement: System workflow section

**Reason:** The standalone marketing workflow section is removed with the old Next.js application.

**Migration:** Product explanations remain available through maintained project documentation until a new marketing-site capability is approved.

### Requirement: Feature and specification sections

**Reason:** The standalone marketing feature and specification sections are removed.

**Migration:** None in this Change.

### Requirement: Analysis output section

**Reason:** The marketing-only analysis-output section is removed.

**Migration:** Real analysis outputs remain available in the Vue workspace and report views under their existing capabilities.

### Requirement: FAQ accordion

**Reason:** The FAQ implementation belonged only to the retired landing page.

**Migration:** None in this Change.

### Requirement: Responsive and accessible interaction basics

**Reason:** This requirement described the removed landing page.

**Migration:** Responsive behavior for the maintained platform remains covered by `swim-analysis-platform-navigation` and Vue workflow capabilities.

### Requirement: Local verification

**Reason:** Next.js lint, build and local-server verification are no longer applicable.

**Migration:** Vue build and development verification are defined by `official-web-frontend-boundary`.
